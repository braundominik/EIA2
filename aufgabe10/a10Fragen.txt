homogen: nur ein Datentyp
heterogen: unterschiedliche Datentypen
Assoziative: Verwendung von Schlüsseln um die enthaltenen Elemente anzusteuern
Es ist hier so, dass sowohl der Schlüsselwert vom Typ "string" sein muss als auch der Wert der zum jeweiligen Schlüssel gehört.

2. Parsed Query Teil der URL und gibt falls true gesetzt ein Objekt der parse Methode des querystring Moduls zurück.
Diese Parse Methode verwandelt den String in in eine Sammlung aus Schlüsseln und Werten.

3. Durchläuft die Eigenschaften eines Objects und weißt diese der angegebenen Variable zu.
In konkreten Fall wird über die Schlüssel des von der Parse Methode erzeugten Objektes iteriert.

4. Standardmäßig blockieren Browser cross-origin HTTP requests durch Skripte. Daher muss die Domain an die der request gesendet wird dies explizit erlauben
indem im Header mit "Access-Control-Allow-Origin" Anfragen erlaubt werden.

6. AJAX - Asynchronous JavaScript and XML
Gewährt einen flüssigen Ablauf z.B wenn Formulare versendet werden. Der User kann während der Verarbeitung des Formulars
weiterhin mit der Seite interagieren. Ein Neuladen der Seite ist nicht erforderlich.
Hierfür wird das XMLHttpRequest object verwendet um mit dem server zu kommunizieren. 

Zu XMLHttpRequest
Das Objekt kann ohne die Seite neu laden zu müssen mit einem Server kommunizieren.
Die Antwort kann ebenfalls über das Objekt im Hintergrund ohne Neuladen entegen genommen werden.
